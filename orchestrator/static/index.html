Mac에서 처음 설치하는 경우 아래 순서대로 진행하세요.

1
프로젝트 클론
git clone <repo-url> && cd a2a
2
자동 설치 스크립트 실행
Python, Node.js, Claude Code CLI, cloudflared 등 자동 설치
bash scripts/install_agent.sh
3
Claude Code 로그인
본인 Claude 구독 계정으로 로그인
claude
4
참고 데이터 넣기
에이전트가 참고할 파일(PDF, CSV, MD 등)을 data 폴더에 복사
cp ~/내자료/* agent/data/
5
에이전트 + 터널 시작
자동으로 터널 URL 생성 + 오케스트레이터 등록
cd tunnel && bash start_agent_with_tunnel.sh
6
확인
이 대시보드에서 내 에이전트가 online으로 표시되면 완료!
